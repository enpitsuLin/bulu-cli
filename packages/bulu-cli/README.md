# @bulu-cli/core

`@bulu-cli/core` provides the `bulu` command-line tool for managing local Ethereum, Tron, and TON wallets.

## Install

```bash
pnpm add -g @bulu-cli/core
```

## Usage

```bash
bulu config init
bulu wallet create
bulu wallet import
bulu wallet list
bulu wallet info <wallet>
```

## Features

- Initialize and manage local CLI config
- Create, import, export, and delete wallets
- Inspect wallet metadata stored in the local vault
